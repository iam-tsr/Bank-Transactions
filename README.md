# Repository Name: Bank Transaction System

## Overview:
This repository contains a simple C++ program for a basic banking system. The program allows users to perform transactions such as depositing money, withdrawing money, checking balance, and transferring money between accounts.

## Contents:
- `main.cpp`: Contains the main C++ code for the banking system.
- `README.md`: This readme file providing information about the repository.

## How to Use:
1. Clone the repository to your local machine.
2. Compile the `main.cpp` file using a C++ compiler.
3. Run the compiled executable file.
4. Follow the prompts to perform transactions.

## Features:
- **Deposit**: Allows users to deposit money into their account.
- **Withdraw**: Allows users to withdraw money from their account, given that there is sufficient balance.
- **Balance Check**: Allows users to check their current account balance.
- **Transfer**: Allows users to transfer money between accounts (not implemented in this version).

## Usage:
- Upon running the program, users are presented with a menu displaying transaction options.
- Users can choose from options to deposit money, withdraw money, or check balance.
- The program ensures proper validation of transactions and provides feedback accordingly.
